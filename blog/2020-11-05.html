Nouvelle organisation pour les paramètres

Enfin du nouveau

J'ai pris pas mal de retard sur le plan que j'avais à l'origine pour préparer ciboulette à la "saison marketing" cet hiver. J'ai pu finir la réorganisation des paramètres, qui ont maintenant un sous menu, pour aider à la compréhension. J'espère que ça rends les paramètres plus clairs.

En résumé : 

  • Profile vendeur : votre description pour les clients
  • Lien d'invitation : formulaire à intégrer à votre page facebook / site web pour que que les visiteurs d'ajoutent automatiquement a votre mailing liste.
  • Mail d'invitation : paramètrage de l'expérience des nouveaux clients
  • Emailing : connexion au compte Gmail pour l'instant
  • Notifications : activer les notifications web
  • Performance : toutes les options pour rendre l'application plus fluide sur un vieux PC
  • Exports : exporter vos données de vente en xlsx

Les modes de paiement utilisés sont dans les options de la page paiement.
