Un petit détail proposé par Le Petit Moulin

Hello

Vous pouvez maintenant filtrer l'export de compta par rendez-vous. 

Seuls les rendez vous présent dans la semaine active sont proposés, donc si vous avez supprimé des rendez vous dans le passé vous ne pourrez pas ensuite les choisir comme filtre. Le filtre est basé sur l'id du rendez-vous, donc si vous avez renommé un rendez-vous il sera toujours considéré comme le même rendez-vous.
