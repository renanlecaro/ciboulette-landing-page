Nouvelles fonctions

Notifications web, liste de clients ayant commandé un produit

Quelques petites améliorations récentes:

  • Les noms de clients sont maintenant dans une couleur représentant leur statut. Vert/Rouge/Gris/Bleu pour Créditeur/Débiteur/Désinscrit/Normal
  • Vous avez une option dans les paramètres pour recevoir les notifications de cyber sur votre ordinateur ou tel, même quand l'appli est fermée. Je vais essayer de limiter les notifications inutiles, parce que pour l'instant je notifie pour un peu tout ce qu'il se passe, même s'il n'y a rien de spécial a faire de votre coté.
  • Dans la page d'accueil, je liste les clients inactifs, vous pouvez maintenant cliquer sur leur nom pour voir les détails de ce client
  • Dans la page de préparation, une petite fonction "cachée" a été ajoutée. Si vous cliquez sur l'entête "Client" une fenêtre s'ouvre avec la liste des emails des clients ayant commandé ce produit.
  • Vous pouvez maintenant consulter l'historique de plantons de ciboulette, sur la page suivante (vous indique les périodes ou l'appli était offline) : https://uptime.ciboulette.net/
  • J'ai maintenant une instance de test, un serveur qui est comme celui de production mais un peu moins performant, et me permet de vérifier qu'un déploiement sur digitalocean se passe bien avant de le lancer "en vrai".
